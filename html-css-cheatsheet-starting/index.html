What I learned so far in HTML

Table tags,names and their description

Tag Name Description
<!Doctype html> Document Type Html Browser declaration
<head> Head The head element and links to other files such as CSS
<body> Main Body Main body of the Html file that contains all the code
<footer> Footer A footer element typically contains ownership information

More information


<p> Paragraph This tag formats any text between the opening tag and the closing tag.
<h2> Heading Tag Using this tag will format any text between the opening tag and the closing tag as a Heading 2 (a type of subheading.)
<b> Bold Tag This tag will format any text between the opening tag and the closing tag as bold.
<u> Underline Tag This tag will format any text between the opening tag and the closing tag as underlined.
<a> Anchor Anchor tag allows to anchor a link whether its to another section of the webpage or a different site
